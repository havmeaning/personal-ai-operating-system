## Overview

This repository hosts the research paper, datasets, and supporting materials for a comparative evaluation of Agent Operating System (AgentOS) frameworks as applied to personal-scale knowledge work. The study examines how modern AI agent orchestration frameworks can serve as foundational infrastructure for a *Personal AI Operating System* — a unified cognitive workspace where autonomous agents assist with memory, planning, research, and decision-making.

---

## Research Questions

1. **Which AgentOS frameworks are best suited for personal-scale knowledge work in 2026?**
2. **What are the architectural trade-offs between agent autonomy, user control, and resource efficiency at the personal scale?**
3. **How do current frameworks address memory persistence, task decomposition, and multi-agent coordination?**
4. **What design principles should guide the construction of a Personal AI Operating System?**

## Key Concepts

| Term | Description |
|------|-------------|
| **PAIOS** | Personal AI Operating System — a unified AI-powered workspace for knowledge work |
